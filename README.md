# Meteor-ChatRoom
####使用Meteor.js寫出的多房間聊天室，使用者可以自由地新增房間，每個房間都是一個多人聊天室


##使用方法
請先安裝meteor.js
> curl https://install.meteor.com/ | sh

下載此專案
> git clone https://github.com/coldbirdOuO/Meteor-ChatRoom.git

打開終端機，在檔案目錄下，輸入meteor
> meteor
